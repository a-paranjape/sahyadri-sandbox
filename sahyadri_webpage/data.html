Data Products

Comprehensive data products for cosmological analysis, available through our public data portal.

Particle Snapshots

Full particle data in custom compressed format (3× reduction, <1% accuracy loss). Positions encoded at 8-bit, velocities at 12-bit precision with subsample partitioning.

HDF5Compressed
Halo Catalogs

Rockstar phase-space halo catalogs with quality control flags. Includes mass, position, velocity, concentration, spin, and merger tree links.

ASCIIHDF5
Matter Power Spectra

Pre-computed P(k) for all snapshots and parameter variations. Validated against Halofit predictions with excellent agreement.

ASCIINumPy
Halo Mass Functions

Differential and cumulative halo mass functions across all snapshots. Compared with Tinker fitting functions showing excellent consistency.

ASCIINumPy
Voronoi Volume Functions

VVF measurements probing cosmic web geometry and void statistics. Demonstrated sensitivity to cosmological parameters, especially Ωₘ.

NumPy
Analysis Tools & Scripts

Python analysis scripts, reading routines for compressed formats, example notebooks, and pipeline documentation.
